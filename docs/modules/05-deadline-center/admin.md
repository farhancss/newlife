# Deadline Center — Admin

**FSD:** §7

## 1. Purpose

Configure global and per-season deadline templates; override per student if needed.

## 2. User stories

| ID | Story |
|----|-------|
| A-DL1 | As an admin, I set season deadline dates by type. |
| A-DL2 | As an admin, I override a deadline for individual student. |

## 3. Admin features

| Feature | Detail |
|---------|--------|
| Season config | Academic year / move-in season |
| Deadline templates | Type + date + description |
| Per-student override | Extension with reason |
| Bulk assign | Apply templates to cohort |

## 4. Acceptance criteria

- [ ] Template changes apply to new students in season.
- [ ] Override visible on student deadline view.
- [ ] Audit log for override actions.
