# Deliveries — Student (view only)

**FSD:** §9, §18.2

## 1. Purpose

Students see scheduled dorm delivery date/window and confirmation — no separate nav required for MVP.

## 2. Visibility surfaces

| Surface | Content |
|---------|---------|
| Student dashboard | Upcoming delivery snippet |
| My Move | "Scheduled for Dorm Delivery" / "Out for Delivery" states |
| Notifications | Delivery scheduled, out for delivery, delivered |

## 3. Post-delivery

Student receives:
- Delivery confirmation timestamp
- Delivery photos (module 15)
- Status → Delivered to Dorm

## 4. Acceptance criteria

- [ ] Scheduled delivery visible on My Move when status applies.
- [ ] Student notified when delivery scheduled and completed.
- [ ] Delivery photos viewable after completion.
