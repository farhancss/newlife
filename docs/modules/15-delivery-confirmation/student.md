# Delivery Confirmation — Student

**FSD:** §18.2

## 1. Purpose

Proof of dorm delivery builds customer confidence.

## 2. Student receives

| Item | Detail |
|------|--------|
| Notification | Delivery completed |
| Timestamp | Delivered at datetime |
| Photos | Package placement / dorm delivery photos |
| Status update | Container → Delivered to Dorm |

## 3. View location

- My Move timeline (completed state)
- Notification deep link → delivery detail modal
- Optional: photo gallery on My Move

## 4. Acceptance criteria

- [ ] Student sees delivery confirmation within 1 hour of admin completion.
- [ ] Photos display in gallery (customer_visible only).
- [ ] Timestamp shown in student timezone.
