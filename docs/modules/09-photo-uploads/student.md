# Photo Uploads — Student

**FSD:** §10.1, §10.3–§10.4

## 1. Purpose

Students document exterior container condition for damage claims.

## 2. User stories

| ID | Story |
|----|-------|
| S-P1 | As a student, I upload exterior photos per container. |
| S-P2 | As a student, I see how many photos I can upload based on tier. |

## 3. Rules

| Rule | Detail |
|------|--------|
| Type | Exterior only — no interior photos |
| Limit | Per package tier (configurable, e.g. 3–5 per container) |
| Format | JPEG/PNG, max 5MB each |
| Disclaimer | "Failure to upload photos may impact damage claim processing" |

## 4. UI location

- My Move page — per container upload section
- Optional reminder in action items

## 5. Acceptance criteria

- [ ] Upload respects per-container cap.
- [ ] Disclaimer acknowledged before first upload.
- [ ] Photos viewable in gallery per container.
- [ ] Delete allowed before return shipment (configurable).
