# Photo Uploads — Admin / Staff

**FSD:** §10.2

## 1. Purpose

Staff upload photos at hub intake and dorm delivery.

## 2. Upload contexts

| Context | Uploader | When |
|---------|----------|------|
| Hub intake | Admin | Container received at hub |
| Dorm delivery | Admin/field | Delivery completion (module 15) |

## 3. Admin features

- View all photos for container
- Upload on behalf of operations
- Photos visible to student when marked customer_visible

## 4. Acceptance criteria

- [ ] Admin upload on container detail and delivery screens.
- [ ] Student sees customer-visible photos only.
- [ ] Storage paths secured with authorization.
